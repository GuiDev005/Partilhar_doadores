## 💡 O que é o Partilhar?

O **Partilhar** é um projeto social localizado no Recife, PE, que oferece apoio psicológico, doação de roupas, alimentos e realiza diversas outras atividades beneficentes para a população mais necessitada.

Para apoiar e organizar essa iniciativa, desenvolvemos o **Sistema Partilhar**. Através da nossa aplicação, os doadores podem se cadastrar, definir por quanto tempo desejam contribuir e receber lembretes automáticos pelo WhatsApp. Do lado administrativo, o sistema facilita a gestão verificando mensalmente quem está ativo e gerando relatórios com o valor total arrecadado.

## ⚙️ Funcionalidades

| ID | Nome | Descrição |
|---|---|---|
| UC01 | Cadastro de Doador | O doador preenche seus dados e se registra no sistema |
| UC02 | Tempo de Compromisso | O doador define por quanto tempo pretende contribuir |
| UC03 | Registro na Planilha | O sistema salva automaticamente os dados do doador após o cadastro |
| UC04 | Verificação Mensal | Rotina automática que verifica o status de cada doador todo mês |
| UC05 | Notificação via WhatsApp | Envio automático de mensagens aos doadores via API do WhatsApp |
| UC06 | Relatório Mensal | Relatório consolidado com dados dos doadores e valores arrecadados |
